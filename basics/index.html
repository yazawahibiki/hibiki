    数学建模论文

    # 数学建模论文

    # 1.问题提出

    随着我国经济的高速发展，城市空间环境复杂性急剧上升，各种事故灾害频发，安全风险不断增大，消防救援队承担的任务也呈现多样化、复杂化的趋势。现在需要根据我国某地方四年来发生火灾等需要消防救援的事故的发生次数对于消防队员的安排进行规划。

    # 2.模型假设

    • 假设一：消防人员不会因为某些原因减少。
    • 假设二：每个消防站向每个区域的出警速率是相同的
    • 假设三：没有无效的出警

    # 3.问题分析

    # 3.1 问题一

    根据问题一的说明：首先处理附件的数据，得到每日不同时间段的数据，再通过c语言编写程序计算权重及根据权重进行人员分配，得到人员规划模型。

    # 3.1 问题二

    根据问题二的说明：先将前四年的数据作为基础，判断2020年数据的有效性，分析2020年是否要加入到预测模型中，再通过该模型预测2021年防救援出警次数的次序并写入表1。

    # 3.1 问题三

    根据问题三的说明：按月份统计5年每类事件在每一个月份的出警总次数，再用5年每类事件在每一个月份上的出警次数的均值进行数据拟合。

    # 3.1 问题四

    算出2016年1月1日到2020年12月31各类事件的事件密度, 做出三维图来表示各个城市之间各个事件之间事件密度的关系, 通过三维图找到不同区域相关性最强的事件类别。

    # 3.1 问题五

    根据问题五的说明：按区域统计并计算出5年每类事件在15个区域的发生的事件密度和15个区域对应的人口密度，建立相关的系数模型，并讨论两者是否有关联。

    # 3.1 问题六

    根据问题六的说明，需要我们考虑各种因素建立数学模型，确定新建一个消防站以及未来的消防站建设。构造优化模型，计算出各个城市之间的最短路径，构建一个动态规划模型来长期的考虑每次的下一个消防站应该建设在哪里。
